# Project Brief: Little Stars

## Overview
Little Stars is a kindergarten and preschool website for children aged 1-6, located in Bakirkoy, Istanbul. The site has 7 pages with a warm, child-friendly design using pastel colors, soft shapes, and rounded corners.

## Target Audience
- Parents looking for daycare/preschool in Bakirkoy area
- Children aged 1-6 years

## Core Requirements
- 7 pages: Home, Programs, Teachers, Gallery, Parent Info, About, Registration
- Pastel color scheme with warm white base
- Nunito font throughout
- Rounded corners everywhere (16-24px for cards, pill buttons)
- Floating decorative SVG animations
- Mobile responsive
- MEB approved badge in footer

## Tech Stack
- Next.js 16 with App Router
- TypeScript
- Tailwind CSS v4
- GSAP, Framer Motion (available)
- Pure CSS animations for floating shapes
